# Анализ удержания пользователей Яндекс Афиши

Проект посвящён исследованию факторов, влияющих на повторные покупки пользователей сервиса Яндекс Афиша.

В ходе работы были выполнены:
- выгрузка данных из PostgreSQL;
- предобработка данных;
- построение профиля пользователя;
- исследование факторов удержания;
- корреляционный анализ признаков пользователей.

Используемые инструменты:
- SQL
- Python
- Pandas
- Matplotlib
- Seaborn
- PhiK
